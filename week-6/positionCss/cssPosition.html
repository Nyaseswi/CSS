Static Position

It is the default position.If we do not specify the position of HTML elements, the broweser automatically adds the position property and sets it to static

Relative Position

This will move the element with the class container to right by 20px, relative to its original position. Attributes left , top

Absolute Position

Behaves like the element is not even there nothing changes when applied absolute property. Use attributes like top,left,right,bottom

Fixed Position

The fixed value is absolute as it can help you position an element anywhere relative to the document, however, this value is unaffected by scrolling.It generally means fixed to the view port.It does not scroll with the page

Sticky Position

Element will be sticked at the specified position as given
